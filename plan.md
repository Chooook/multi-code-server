### Шаги

- Установка зависимостей
- Создание и настройка необходимых директорий
- Создание шаблонов конфигураций
- Установка скриптов с пользовательским доступом:
  - интерактивное управление сервисами
  - пересоздание конфигураций
  - смена пароля code-server
  - просмотр логов (каких сервисов?)
- Установка root скриптов:
  - создание конфигураций сервисов (с учетом переменных окружения, путей, зависимостей, D-bus и логирования)
  - выделение пользовательских портов
  - очистка конфигураций и отключение сервисов определенного пользователя
  - получение статуса сервисов пользователя (наличие конфигураций, текущая активность systemd сервисов и nginx прокси)
  - просмотр логов определенных пользователей (всех сервисов на выбор)
- Создание сервиса на таймере для автоматического создания конфигураций сервисов пользователя:
  - Настройка лингеринга
  - systemd сервис code-server
  - nginx прокси сервис
  - systemd сервис мониторинга активности (для остановки/запуска в зависимости от активности)
  - Создание инструкций:
    - к code-server и его управлению через пользовательские скрипты
    - по настройке ssh
    - по простому подключению к терминалу и пробросу портов по ssh (личная машина)
